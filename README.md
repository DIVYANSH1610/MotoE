## 🧠 Overview

**AutoIntel Garage** is a full-stack, AI-powered automotive platform engineered like a digital racing cockpit.

It combines:

* rich car exploration
* AI-assisted car understanding
* used car decision support
* a premium, performance-inspired UI
* user-authenticated personal garage features

Instead of feeling like a plain catalogue, the platform is designed to feel like a **high-performance control system for car enthusiasts**.

---

## 🎮 Core Modules

### 🏎️ Car Exploration Engine

* Dynamic car listing and detail pages
* Slug-based routing for clean URLs
* Specs, story, design notes, history, and engineering insights
* Rich gallery and lightbox experience

### 🤖 AI Garage Assistant

* Ask questions about any specific car
* Get model-aware responses
* Explore design, performance, reliability, practicality, and brand character

### 🚗 Used Car AI Advisor

* Submit used car details
* Get structured AI analysis
* Receive smarter buy / caution / avoid style guidance

### ❤️ Personal Garage

* Save favourite cars
* Maintain a personalized collection
* Extend the platform beyond passive browsing

### 🎨 Racing UI Experience

* Glassmorphism-inspired layout
* Motion-rich interactions
* Premium dashboard aesthetics
* Built to feel energetic, immersive, and modern

## 🛠️ Local Setup

### Clone the repository

```bash
git clone https://github.com/YOUR-USERNAME/YOUR-REPO.git
cd YOUR-REPO
```

### Frontend setup

```bash
cd frontend
npm install
npm run dev
```

### Backend setup

```bash
cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
